Hallo!

Mijn naam is Patrick Lensen.

Aspirant Front End Developer, met de nadruk op HTML, CSS en Javascript en zijn frameworks React.js en Vue.js. En graag gebruik makend van Gulp en Webpack om ontwikkelen een beetje te automatiseren.

HTML/CSS/SASS programmeer ik via het BEM principe voor een heldere structuur in niet alleen code maar ook bestanden architectuur, en uiteraard ga ik uit van mobile first.

Met JavaScript maak ik gebruikt van:

  • Programmeren Vanilla JS in modules
  • ES6
  • jQuery
  • React Framework
  • Angular Framework
  • Vue Framework
  • Automation tools zoals :Gulp, Webpack, Babel
Verder verdiep ik me graag ook in PHP en het framework Lavarel, als ook Unity
